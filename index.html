Full Site Rebuild · Featured Case Study

Rebuilding a 30-year firm — client: Adisa Communications

I rebuilt a strategic-communications firm's 30-year legacy WordPress site into a modern React 19 / Vite single-page app on Cloudflare's global edge. My "scrape & rebuild" migration preserved three decades of their content and SEO link structure while converting the database to immutable data (zero DB lookups → instant loads). Removing the server and database eliminated ~99% of common attack surface (SQL injection, XSS, plugin exploits); I added Framer Motion and a persistent background-video engine for a premium, app-like feel. Outcome: a faster, maintenance-free, near-zero-cost platform — architected, built, and deployed end to end by me.
